JOINS
1.CARTESIAN/CROSS JOIN:-
EACH AND EVERY DATA OF TABLE1 WILL BE COMBINED WITH EACH AND EVERY TABLE OF TABLE2.
2.INNER JOINS:-
YOU WILL ONLY GET MATCHED RECORDS.
WHEN YOU ARE EXPLICITLY WRITING THE  JOIN NAME YOU HAVE TO USE ON CLAUSE. THIS IS ASCII SYNTAX.
ASCI SYNTAX -
EX- SELECT *
FROM EMP INNER JOIN DEPT
ON EMP.DEPTNO = DEPT.DEPTNO;

ORACLE SYNTAX-(DONE IMPLICITLY)
SELECT *
FROM EMP,DEPT
WHERE EMP.DEPTNO = DEPT.DEPTNO;

3.OUTER JOINS-----1.LEFT [OUTER] JOIN
                  2.RIGHT[OUTER] JOIN( GIVE UNMATCHED TABLE OF THE RIGHT SIDE OF THE EQUATION )

                  3.FULL[OUTER] JOIN
4.SELF JOIN
5.NATURAL JOIN
